# INFO 2602 Lab 1: Python and FastAPI
Welcome to the repository for INFO 2602 Lab 1! This lab provides a brief introduction to Python syntax and FastAPI development.

## Objectives
This lab will cover:

*   Basic Python syntax
*   Understanding how FastAPI works
*   Exploring Routes
*   Route Parameters
*   Query Parameters

## Getting Started
1.  **Environment Setup:** Ensure you have Python installed. You will be using VSCode for development.
2.  **Create a virtual environment:** `python -m venv venv`
3.  **Activate the virtual environment:**
    *   **Windows:** `venv\Scripts\activate`
    *   **macOS / Linux:** `source venv/bin/activate`
4.  **Install dependencies:** `pip install -e ".[dev]"` 
5.  **Start the development server:** `fastapi dev`. If this gives an error, try `fastapi dev --port 9090` 

## Accessing Your Application

Once you've set up your FastAPI application, you can access it at `http://localhost:9090`
